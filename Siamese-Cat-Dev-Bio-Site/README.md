# Siamese Cat Dev Bio Site

An animated personal bio site built with React, TypeScript, Vite, Framer Motion, and Lucide React.

## Run locally

```bash
npm install
npm run dev
```

## Production build

```bash
npm run build
npm run preview
```

The site uses only the supplied Siamese Cat Dev and DJAI Academy image assets. The interactive hero mascot is a transparent, wordless cutout derived from the supplied logo.
