FAQ

Questions about image resizing.

Can the tool make an image exactly 100 KB?

The compressor aims for the selected size or slightly below it. The final size can vary by a few KB because every image compresses differently.

Will my PNG transparency be preserved?

Yes, when the output format is PNG or WebP. Converting a transparent image to JPG adds a white background because JPG does not support transparency.

Does reducing the file size reduce image quality?

Target-KB compression may lower quality or dimensions. Percentage and dimension modes let you control the image size directly while using a high-quality export.

Are my images uploaded to DJAI?

No. The tool works locally in your browser and does not send the image to DJAI or Siamese Cat Dev.
