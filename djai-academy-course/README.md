# DJAI Academy Next.js Website

This is a rebuilt Next.js landing page for DJAI Academy's AI Masterclass.

## Run locally

```bash
npm install --cache .npm-cache
npm run dev
```

Open `http://localhost:3000`.

## Production build

```bash
npm run build
npm run start
```

## What was fixed

- Unified the workshop as a 1-day, 8-hour hands-on masterclass.
- Standardized contact email to `contact@djai.academy`.
- Changed the FAQ label to `FAQs` consistently.
- Replaced unsourced percentage claims with clearer creator journey messaging.
- Reframed the app mockup as an example product interface rather than a generic shopping app promise.
